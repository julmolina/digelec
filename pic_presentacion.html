Microcontroladores PIC

Los microcontroladores PIC (conocidos simplemente como PIC) son circuitos integrados programables que pueden ejecutar un sinnúmero de tareas y procesos de forma automática. El programa se escribe en un computador en un ambiente de programación adecuado (por ejemplo en lenguaje C) y luego se transfiere como código binario a la memoria de programa del microcontrolador para su ejecución en el momento de encendido del dispositivo. Los microcontroladores PIC son fabricados por la empresa Microchip y constan básicamente de una CPU, espacios de memoria para el programa (ROM), memoria para los datos (RAM), y varios periféricos o puertos para la interacción con el mundo externo (lineas de entrada/salida), todo esto encapsulado dentro de un solo chip de silicio.

Existen diferentes familias de microcontoladores, tales como: Intel, Altair, Siemens, Basic Stamp, Microchip, National Semiconductor, Motorola, Freescale, Texas Instruments, Atmel, entre otros.

Como modo efectos de estudio se tomará como referencia el microcontrolador PIC16F887, el cual pertenece a la familia de los microchips. Todos los conocimientos adquiridos a partir de esta referencia, puede ser implementada a otro microcontrolador de la familia PIC16 o PIC18. Estos microcontroladores pertenecen a la familia RISC fabricados por Micrichip Tecnology Inc y posee una arquitectura Harvard caracterizada por la independencia entre la memoria de código y la de datos.

¿Por qué usar un Microcontrolador PIC?

El microcontrolador PIC es popular por su combinación de bajo costo, flexibilidad, eficiencia energética y simplicidad en el software. Su arquitectura le permite ser una opción ideal tanto para principiantes como para proyectos avanzados en diversas áreas como la industria, transporte, aviación, medicina y comunicaciones.

Ventajas de utilizar un microcontrolador PIC

    *Bajo costo: Los microcontroladores PIC son económicos, lo que los hace ideales para la fabricación de productos a bajo costo.

    *Diseño flexible: Gracias a su arquitectura Harvard y la posibilidad de conectar una gran cantidad de periféricos, los microcontroladores PIC se pueden utilizar en una amplia gama de aplicaciones, desde la industria hasta la medicina.

    *Software sencillo: La uniformidad del software PIC facilita la depuración y la resolución de problemas, lo que es especialmente beneficioso para los principiantes.

    *Bajo consumo de energía: Los microcontroladores PIC consumen poca energía, lo que los hace adecuados para dispositivos pequeños que requieren eficiencia energética.

    *Rápida ejecución: Las instrucciones en los PIC suelen ejecutarse en un solo ciclo de reloj (1μS con un reloj de 4MHz), lo que permite operaciones rápidas.

Limitaciones del uso de microcontroladores PIC

    *Programabilidad limitada: Comparado con microcontroladores más grandes, los PIC tienen menos características programables, lo que puede dificultar la personalización del hardware para necesidades específicas.

    *Recursos limitados: Los PIC son de bajo costo, lo que también significa que tienen menos recursos asignados, lo que puede extender los tiempos de desarrollo y afectar el rendimiento en proyectos más complejos.

    *Instrucciones limitadas: Los microcontroladores PIC de la familia 16F, por ejemplo, tienen solo 35 instrucciones debido a su arquitectura RISC, lo que puede complicar la vida del programador. Sin embargo, usar un lenguaje de programación como C con un compilador como PIC C Compiler CCS C puede simplificar el desarrollo.

Lenguajes de programación para PIC

El microcontrolador ejecuta el programa cargado en la memoria Flash. Esto se denomina el código ejecutable y está compuesto por una serie de ceros y unos, aparentemente sin significado. Dependiendo de la arquitectura del microcontrolador, el código binario está compuesto por palabras de 12, 14 o 16 bits de anchura. Cada palabra se interpreta por la CPU como una instrucción a ser ejecutada durante el funcionamiento del microcontrolador. Todas las instrucciones que el microcontrolador puede reconocer y ejecutar se les denominan colectivamente Conjunto de instrucciones. Como es más fácil trabajar con el sistema de numeración hexadecimal, el código ejecutable se representa con frecuencia como una serie de los números hexadecimales denominada código Hex.

Existen diversos lenguajes de programación (alto nivel, bajo nivel y nivel medio) que se pueden utilizar para programar el microcontrolador PIC, estos son:

* Lenguaje ensamblador (Assembler): es un lenguaje de programación de bajo nivel, y se traduce a código máquina mediante un programa ensamblador. El lenguaje ensamblador actúa como un intermediario entre el código máquina y los lenguajes de alto nivel, ofreciendo una comunicación más directa con el hardware. Cada familia de procesadores posee su propio lenguaje ensamblador, vinculado estrechamente a su arquitectura específica.

* Lenguaje C: es un lenguaje de nivel medio bajo, así que no requiere de muchos recursos pero ofrece una buena respuesta y una ejecución rápida de programas. El lenguaje de programación C ha sido una piedra angular en el desarrollo de software durante más de cuatro décadas. Ha influido en numerosos lenguajes modernos y sigue siendo una elección popular para sistemas críticos.

* Lenguaje C++: es un lenguaje de alto nivel, C++ también es un ejemplo de lenguaje de programación compilado, multiparadigma, principalmente de tipo imperativo y orientado a objetos, incluyendo también programación genérica y funcional. Un programa escrito en un lenguaje imperativo es un conjunto de instrucciones que indican al computador cómo realizar una tarea.

* Python: Python es un lenguaje de programación de alto nivel ampliamente utilizado en las aplicaciones web, el desarrollo de software, la ciencia de datos y el machine learning (ML). Los desarrolladores utilizan Python porque es eficiente y fácil de aprender, además de que se puede ejecutar en muchas plataformas diferentes.

* Micropython: es una implementación software del lenguaje de programación Python 3, escrita en C, y que está optimizada para poder ejecutarse en un microcontrolador. MicroPython es un compilador completo del lenguaje Python y un motor e intérprete en tiempo de ejecución, que funciona en el hardware del microcontrolador.

* Java: es un lenguaje de programación de alto nivel ampliamente utilizado para codificar aplicaciones web. Ha sido una opción popular entre los desarrolladores durante más de dos décadas, con millones de aplicaciones Java en uso en la actualidad. Java es un lenguaje multiplataforma, orientado a objetos y centrado en la red que se puede utilizar como una plataforma en sí mismo.

Programador PICkit

Son dispositivos que permiten depurar y programar de forma rápida y sencilla los microcontroladores PIC, estos dispositivos utilizan el entorno de desarrollo integrado MPLAB desarrollado por Microchip el cual trabaja sobre plataformas de Windows.

Existen diversas versiones de PICkit tales como el PICkit2, PICkit3, PICkit3.5 y PICkit4, sin embargo la versión más conocida y utilizada es el PICkit3, por tal motivo utilizaremos esta versión para llevar a cabo los laboratorios propuestos.
