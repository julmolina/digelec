Laboratorio 1 - Control de un LED con ESP32

Introducción

Este laboratorio te enseñará cómo encender y apagar un LED utilizando el GPIO del ESP32. Ideal para empezar a interactuar con hardware y aprender lo básico.

Justificación

El control de LEDs es una tarea fundamental al trabajar con microcontroladores, ya que permite aprender sobre la manipulación de pines y la interacción con dispositivos externos.

Objetivos

  • Entender cómo manipular los GPIOs del ESP32.
  • Controlar un LED de manera programática.

Problema

¿Cómo se puede controlar un LED utilizando el ESP32 para que se encienda y apague según instrucciones del código?

Requerimientos

  • ESP32
  • LED
  • Resistencia de 220Ω
  • Cables para conexión
  • Breadboard
Código para el Control del LED
#include <Arduino.h>
int ledPin = 2; // Pin donde está conectado el LED

void setup() {
    pinMode(ledPin, OUTPUT); // Configuramos el pin como salida
}

void loop() {
    digitalWrite(ledPin, HIGH); // Encendemos el LED
    delay(1000); // Esperamos 1 segundo
    digitalWrite(ledPin, LOW);  // Apagamos el LED
    delay(1000); // Esperamos 1 segundo
}
