Bibliografía - ESP32

Encuentra aquí las principales referencias para trabajar con la ESP32, tanto de libros como de recursos online.

Libros

  • Kolban, N. (2017). Kolban's Book on the ESP32 & ESP8266. Leanpub.
    Disponible en: https://leanpub.com/kolban-ESP32
  • Santos, R., & Alves, S. (2020). ESP32 Programming for the Internet of Things: A Hands-On Guide. Packt Publishing.
  • Vasiliev, Y. (2020). MicroPython Projects: A Do-It-Yourself Guide for Embedded Developers. Packt Publishing.

Páginas Web

  • Espressif Systems - ESP32 Documentation
    Documentación oficial de Espressif Systems para el ESP32. Ideal para desarrolladores que quieren entender a fondo la arquitectura y las APIs.
  • Random Nerd Tutorials - Proyectos con ESP32
    Una de las mejores páginas para guías y proyectos con ESP32. Excelente para principiantes.
  • Arduino Reference - ESP32 Libraries
    Lista de bibliotecas para el ESP32 compatibles con el entorno de Arduino.
  • MicroPython - The MicroPython Language
    Información sobre MicroPython, un lenguaje de programación eficiente para dispositivos como el ESP32.
  • DigiKey - Cómo seleccionar y usar el módulo ESP32 adecuado
    Guía detallada sobre cómo seleccionar el módulo ESP32 correcto según tus necesidades.
  • TodoMaker - Conociendo al ESP32
    Un artículo introductorio sobre las capacidades y aplicaciones del ESP32, ideal para principiantes.
