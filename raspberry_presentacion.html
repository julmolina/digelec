Raspberry Pi

El Raspberry Pi es un microordenador de placa única (SBC, por sus siglas en inglés) creado por la Fundación Raspberry Pi en el Reino Unido con el objetivo principal de fomentar la enseñanza de la informática básica en las escuelas y países en desarrollo. Desde su lanzamiento en 2012, se ha convertido en una herramienta fundamental no solo en el ámbito educativo, sino también en proyectos de electrónica, desarrollo de software, investigación, y en la creación de soluciones innovadoras en diversos campos.

Características principales del Raspberry Pi:

  • Procesador: Utiliza procesadores ARM de bajo consumo, típicamente con 4 u 8 núcleos dependiendo del modelo.
  • Memoria RAM: Dependiendo del modelo, puede tener desde 512 MB hasta 8 GB de RAM.
  • Almacenamiento: No tiene almacenamiento interno, pero utiliza tarjetas microSD para el sistema operativo y almacenamiento de datos.
  • Sistema Operativo: Su sistema operativo principal es Raspberry Pi OS (anteriormente Raspbian), una versión de Linux basada en Debian, aunque es compatible con otros sistemas operativos como Ubuntu, Windows IoT Core y diversas versiones de Android.
  • Conectividad: Dependiendo del modelo, puede incluir puertos USB, HDMI, GPIO, Ethernet, Wi-Fi y Bluetooth.
  • GPIO: Dispone de un conjunto de pines GPIO (General Purpose Input/Output) que permiten interactuar con una amplia variedad de componentes electrónicos como LEDs, sensores, motores, y más.
  • Aplicaciones: Se utiliza en una gran variedad de proyectos, desde la creación de servidores caseros, sistemas de automatización del hogar, robots, estaciones meteorológicas, hasta proyectos educativos y de entretenimiento.

El Raspberry Pi es muy popular entre los aficionados a la tecnología y los educadores debido a su bajo costo, tamaño compacto, y gran comunidad de soporte.

Raspberry Pi pico

La Raspberry Pi Pico es un microcontrolador diseñado y lanzado por Raspberry Pi Foundation. A diferencia de las computadoras de placa única de la serie Raspberry Pi (como la Raspberry Pi 4 o la Raspberry Pi Zero), que funcionan como pequeñas computadoras con sistemas operativos completos, la Raspberry Pi Pico está diseñada para ser un microcontrolador simple y eficiente en costo, ideal para proyectos de electrónica y programación de bajo nivel.

Características principales del Raspberry Pi pico:

  • Procesador: Dual-core ARM Cortex-M0+ a 133 MHz.
  • Memoria RAM: 264 KB de SRAM y 2 MB de memoria flash integrada.
  • Almacenamiento: Soporte para expandir la memoria flash externa a través de un conector QSPI.
  • Programacion: Compatible con MicroPython, C/C++ y otros lenguajes de programación de microcontroladores.
  • Conectividad: Soporte para UART, SPI, I2C para comunicaciones seriales.
  • GPIO: 26 pines GPIO multifunción, que pueden usarse para digital I/O, PWM, ADC, UART, SPI, I2C, etc. 3 pines ADC de 12 bits para leer señales analógicas.

La Raspberry Pi Pico es una opción versátil y económica para proyectos de electrónica, automatización, robótica, y aprendizaje de programación de microcontroladores.

Partes y componentes de la raspberry Pi

Componentes de la raspberry Pi

  • Entrada Tarjeta SD: Este es el lugar donde se inserta la tarjeta microSD que contiene el sistema operativo y el almacenamiento principal de la Raspberry Pi. Funciona como el disco duro de la Raspberry Pi.
  • Conexión DSI Display: Puerto de conexión para pantallas compatibles con el estándar DSI (Display Serial Interface), como la pantalla táctil oficial de Raspberry Pi.
  • Micro USB Alimentación: Puerto a través del cual se suministra energía a la Raspberry Pi. Requiere un adaptador de 5V/2.5A (según el modelo).
  • Salida Video HDMI: Conector HDMI para la salida de video y audio a monitores o televisores. Permite visualizar la interfaz gráfica de la Raspberry Pi en pantallas externas.
  • Conector Cámara CSI: Puerto de conexión para cámaras compatibles con el estándar CSI (Camera Serial Interface), como la cámara oficial de Raspberry Pi. Se utiliza para proyectos de captura de video e imagen.
  • Salida Sonido/Video RCA Jack: Conector combinado para la salida de audio analógico y video compuesto (RCA). Se utiliza para conectar la Raspberry Pi a televisores antiguos o dispositivos de audio analógico.
  • 40 pin Extended GPIO: Cabecera de pines GPIO (General Purpose Input/Output) que permite la conexión de la Raspberry Pi con una amplia variedad de sensores, actuadores, y otros dispositivos electrónicos. Estos pines permiten realizar proyectos de hardware controlados por la Raspberry Pi.
  • CPU/Chipset: Este es el procesador principal de la Raspberry Pi, que maneja todas las operaciones computacionales. Dependiendo del modelo, puede incluir también una GPU (Unidad de Procesamiento Gráfico) integrada.
  • 4x USB: Cuatro puertos USB estándar para conectar periféricos como teclados, ratones, unidades de almacenamiento, o adaptadores de red.
  • Conector Ethernet: Puerto para conectar la Raspberry Pi a una red mediante un cable Ethernet, proporcionando una conexión a internet por cable o red local.

Partes y componentes de la raspberry Pi pico

Componentes de la raspberry Pi pico

  • USB Micro B Power & Data: Este puerto se utiliza tanto para alimentar la Raspberry Pi Pico como para transferir datos. Es el principal punto de conexión para interactuar con la Pico desde una computadora.
  • Programmable LED: Un LED incorporado que puede ser programado para encenderse o apagarse, o parpadear, proporcionando una forma simple de dar retroalimentación visual en tus proyectos.
  • Boot Selection Button: Un botón que permite seleccionar el modo de arranque. Al mantenerlo presionado mientras se conecta el USB, la Pico entra en modo de almacenamiento masivo para facilitar la programación o actualización del firmware.
  • 2 MByte Quad-SPI Flash: La Raspberry Pi Pico incluye 2 megabytes de memoria flash, utilizada para almacenar el código y los datos del programa que se ejecuta en la microcontroladora.
  • 0.1" Spacing Pads Breadboard Friendly: Los pines de la Pico tienen un espaciado de 0.1 pulgadas, lo que hace que sea compatible con la mayoría de las placas de pruebas (breadboards), facilitando su uso en proyectos de prototipado.
  • RP2040 MCU Silicon: El microcontrolador principal de la Raspberry Pi Pico. Es un chip de doble núcleo con arquitectura ARM Cortex-M0+, que incluye 264 KB de SRAM y funciona a una velocidad de reloj de 48 MHz, con un máximo de 133 MHz. Soporta USB 1.1 como host y dispositivo.
  • Built-in 12-bit Temperature Sensor: Un sensor de temperatura de 12 bits integrado en el microcontrolador, que permite medir la temperatura ambiente sin necesidad de componentes adicionales.
  • SMT friendly Castellated Edges: Los bordes de la placa tienen pads semi-perforados que facilitan la soldadura superficial (SMT), permitiendo su integración en otros circuitos de manera sencilla.
